# NodeAPI
Development of an API rest with nodeJS and ExpressJS

## HowTo

### Récupération du dépôt

> Clone avec SSH

```bash
$ git clone git@github.com:jeremw264/NodeAPI.git
```

> Clone avec HTTPS

```bash
$ git clone https://github.com/jeremw264/NodeAPI.git
```

### .env exemple

```env
NODE_ENV= development

JWT_KEY=<key>

DB_HOST="localhost"
DB_PORT = 27017
DB_NAME = <dbName>

```
